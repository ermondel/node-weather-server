# Node weather server

_Project for Udemy NodeJS course._

### About

Weather forecast for the specified location using the Mapbox API and the Dark Sky API.

### Usage

[Demo](https://demo-node-weather-server.herokuapp.com)

### Install

```
npm install
```

### Technologies

- Node
- Express
- Mapbox API
- Dark Sky API
- Heroku
